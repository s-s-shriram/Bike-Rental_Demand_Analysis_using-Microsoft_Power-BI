# 🚲 Bike Rental Demand Analysis using Power BI
# 📌 Project Overview

This project focuses on analyzing bike rental demand patterns using data visualization techniques. The analysis is performed using Microsoft Power BI to understand how different factors such as time, season, and weather conditions influence bike rental usage.

The goal of this project is to transform raw bike rental data into meaningful insights through interactive dashboards. These insights help identify peak demand periods and support better decision-making for bike rental service management.

# ❗Problem Statement

Urban transportation systems increasingly use bike sharing services to promote eco-friendly and convenient mobility. However, managing bike availability according to demand remains a major challenge.

Demand varies depending on several factors such as:

-  Time of the day	
	 
-  Weather conditions
	 
-  Seasonal changes

Without proper analysis of historical rental data, it becomes difficult to predict peak usage periods and allocate bikes efficiently. Therefore, analyzing bike rental data is essential to understand usage patterns and support better operational planning.

# 📊 Dataset Information

The dataset used in this project is the Seoul Bike Sharing Dataset, which contains historical bike rental data along with environmental and time-related attributes.

📉 Dashboard Features

The Power BI dashboard includes several visualizations such as:
- 📊 Total Bike Rental Count				
- ⏰ Hourly Rental Demand Analysis				
- 🌦 Weather Impact on Bike Rentals				
- 🍂 Seasonal Rental Trends				
- 📈 Overall Rental Distribution

These visualizations help understand customer behavior and identify peak demand periods.

# 📌 Results and Insights

The developed dashboard provides meaningful insights into bike rental demand. The analysis highlights how factors like time of day, seasonal changes, and environmental conditions influence rental patterns.

These insights can help improve resource planning and ensure better availability of bikes during high-demand periods.
